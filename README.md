# UT Exam Quiz

Simple static quiz platform for UT exam practice. The app reads generated question data from `data.js`, separated by course, and runs directly in the browser.

## Local Preview

```sh
python3 -m http.server 5173
```

Then open `http://127.0.0.1:5173/`.
